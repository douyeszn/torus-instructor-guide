# Torus Instructor Guide — RISE Rwanda

A self-contained slide deck that walks facilitators through the OLI Torus instructor dashboard. Each slide covers one feature, showing what it displays, how to interpret it, and what action to take.

## Viewing

Open `index.html` in any browser. Navigate with the arrow buttons, the dot indicators, or the left and right arrow keys.

## Contents

Eleven slides: Dashboard Summary, AI Recommendations, Filters, Student Progress, Student Profiles, Learning Objectives, Learning Objective Details, Assessments, Answer Distributions, Email and Student Outreach, and Data Export.

Built for RISE Rwanda.
